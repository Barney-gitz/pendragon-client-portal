# Pendragon Client Portal

A secure customer portal built with FastAPI, PostgreSQL and Docker.

## Project Goal

This project simulates a real commercial customer portal and is being developed as a professional software engineering portfolio project.

The focus is on backend engineering, authentication, security, infrastructure and modern development practices rather than simply building CRUD functionality.

## Planned Features

### Customer

- Invitation-based account activation
- Secure authentication
- Multi-Factor Authentication (TOTP)
- View company tickets
- Upload files
- Download reports
- Password reset
- Account activity history

### Administrator

- Invite users
- Disable users
- Reset passwords
- Reset MFA
- Manage companies
- View audit logs
- Security dashboard

## Technology Stack

- Python
- FastAPI
- PostgreSQL
- SQLAlchemy
- Alembic
- Docker
- pytest

## Development Philosophy

This project is intentionally built one feature at a time following professional software engineering practices.

Every architectural decision is documented and understood before implementation.
